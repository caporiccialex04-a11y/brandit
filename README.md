# BRANDiT

Clean, modern marketing site for **BRANDiT** — an award-winning branding & digital studio based in Beirut and Dubai.

Content and portfolio imagery are drawn from [brandit.co](http://www.brandit.co/) and the studio’s Instagram presence ([@brand.it.co](https://www.instagram.com/brand.it.co/)).

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
